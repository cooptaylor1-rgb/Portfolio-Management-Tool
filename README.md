### Prerequisites
- Node.js (version 16 or higher)
- npm or yarn package manager

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/cooptaylor1-rgb/Portfolio-Management-Tool.git
   cd Portfolio-Management-Tool
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**
   Navigate to `http://localhost:5173` to see the app in action!

### Building for Production

To create a production-ready build:

```bash
npm run build
```

The optimized files will be in the `dist` directory.

To preview the production build locally:

```bash
npm run preview
```

## üì± How to Use

### Adding an Investment

1. Click the **"+ Add Investment"** button
2. Fill in the investment details:
   - **Name**: Full name of the investment (e.g., "Apple Inc.")
   - **Symbol**: Trading ticker (e.g., "AAPL")
   - **Type**: Choose from stocks, ETFs, bonds, etc.
   - **Quantity**: Number of shares you own
   - **Purchase Date**: When you bought it
   - **Purchase Price**: What you paid per share
   - **Current Price**: Current market price per share
3. Click **"Add Investment"** to save

### Updating Prices

- Click the pencil ‚úèÔ∏è icon next to any investment
- Enter the new current price
- Click the checkmark ‚úì to save

### Viewing Your Portfolio

- **Dashboard Stats**: View key metrics at the top of the page
- **Asset Allocation**: See how your money is distributed across investment types
- **Investment List**: Detailed table showing all your holdings

### Getting Help

Click the **"Help"** button in the header to access:
- Investment type explanations
- Key terminology definitions
- Beginner investment tips
- How-to guide for using the app
